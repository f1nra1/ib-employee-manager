# Система управления сотрудниками отдела ИБ

Курсовой проект по дисциплине "Информационная безопасность"

**Выполнил:** Волынцев А.А., группа АБ-421  
**Университет:** НГТУ

## Описание

Настольное приложение для управления данными сотрудников отдела информационной безопасности. Позволяет вести учёт сотрудников, их должностей, отделов и доступ к информации.

## Технологии

- Java 17
- JavaFX 21
- PostgreSQL 15
- BCrypt (хеширование паролей)
- HikariCP (пул соединений)
- Docker
- Maven

## Функциональность

- Аутентификация и авторизация пользователей
- Защита от подбора паролей (блокировка после 5 неудачных попыток)
- Управление сотрудниками (добавление, редактирование, удаление)
- Управление должностями и отделами
- Разделение прав (администратор/пользователь)
- Экспорт данных в Excel
- **Импорт данных из Excel-файлов (.xlsx)**
- Журнал действий пользователей


## Требования

- Java 17+
- Maven 3.8+
- Docker

## Запуск

```bash
./run.sh
```

Скрипт автоматически:
1. Запустит PostgreSQL в Docker
2. Инициализирует базу данных
3. Соберёт и запустит приложение

## Учётные записи

| Логин | Пароль   | Роль          |
|-------|----------|---------------|
| admin | admin123 | Администратор |
| user  | user123  | Пользователь  |

Пользователи создаются автоматически при первом запуске.

## Безопасность

- Хеширование паролей: BCrypt
- Защита от перебора: блокировка на 15 минут после 5 неудачных попыток
- Параметризованные SQL-запросы
- Журналирование действий пользователей
